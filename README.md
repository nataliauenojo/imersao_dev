# Buscador de Artistas

## Descrição
Este projeto é um buscador de artistas simples, desenvolvido utilizando HTML, CSS e JavaScript. O usuário pode pesquisar por artistas ou músicas e os resultados são exibidos em tempo real.

## Funcionalidades
* **Pesquisa:** Permite pesquisar por artistas ou músicas.
* **Resultados dinâmicos:** Exibe os resultados da pesquisa em tempo real.
* **Links para mais informações:** Cada resultado possui um link para mais informações sobre o artista ou música.

## Como usar
1. **Clone o repositório:**
   ```bash
   git clone [https://www.youtube.com/watch?v=GSswm-2u3Co](https://www.youtube.com/watch?v=GSswm-2u3Co)
   ```
2. **Abra o projeto em um editor de código:**
   Recomendamos utilizar um editor como Visual Studio Code, Sublime Text ou Atom.
3. **Abra o arquivo `index.html` em um navegador.**

## Estrutura dos arquivos
* **index.html:** Arquivo principal da aplicação, contendo a estrutura HTML.
* **style.css:** Arquivo CSS responsável pelo estilo visual da página.
* **dados.js:** Arquivo JavaScript contendo os dados dos artistas (nome, descrição, link, etc.).
* **app.js:** Arquivo JavaScript contendo a lógica da aplicação, incluindo a função de pesquisa.
